== Description ==

Converts links to videos hostet at twitvid.com to embedded players. Bringt embed and download options.

== Installation ==

1. Unpack the zipfile twitvid-X.y.zip
1. Upload folder `twitvid` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to settings -> TwitVid and make it fit your needs
